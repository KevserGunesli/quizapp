Quiz App (English)
A comprehensive classroom and quiz management application developed with Flutter and Firebase infrastructure, featuring multi-role support (Teacher and Student). 
It serves as an interactive platform where teachers can create classrooms and quizzes, and students can participate and track their progress.

-Features
Authentication & Roles
Login/Signup: Secure authentication via Firebase Auth.
Role Management: Distinct interfaces and permissions for Teachers and Students.
-Teacher Module
Classroom Management: Create classes and manage students.
Quiz Creation: Prepare and publish quizzes for specific classrooms.
Analytics: View student statistics and exam results (powered by charts).
-Student Module
Join Classrooms: Enroll in classes created by teachers.
Take Quizzes: Answer assigned quizzes.
Track Results: View exam scores and correct/incorrect answers instantly.
- General Features
Leaderboard: Ranking systems for students.
Profile Management: Edit user profiles.
Visualization: UI enriched with Lottie animations and FL Chart.
